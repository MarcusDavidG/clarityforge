# clarityforge

Clarityforge is a focused workspace for crafting, testing, and refining
Clarity smart contracts on the Stacks blockchain.

The goal is to build well-structured, secure, and reusable contract patterns
while leveraging proper testing and best practices.

## Scope
- Clarity smart contract development
- Unit and fuzz testing
- Security-focused contract design
- Reusable contract primitives

## Tooling
- Clarinet SDK
- Rendezvous (fuzz testing)
- Stacks blockchain

## Status
Under active development.
